# ScreenshotCapture

A Windows application for capturing screenshots with PDF generation capabilities.

## Features

- Screenshot capture functionality
- PDF generation using iTextSharp
- Cryptographic support via BouncyCastle

## Files

- `ScreenshotCapture.exe` - Main executable
- `PDFER.exe` - PDF processing utility
- `iTextSharp.LGPLv2.Core.dll` - PDF library
- `BouncyCastle.Crypto.dll` - Cryptography library
- `icon.ico` - Application icon

## Requirements

- Windows operating system
- .NET Framework (version specified in config)

## Usage

Run `PDFER.exe` to start the application.

## Configuration

Application settings can be modified in `ScreenshotCapture.exe.config`.
